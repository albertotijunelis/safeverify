Everything You Need

From static analysis to ML-powered classification — a complete toolkit for malware researchers and SOC analysts.

Static Analysis

PE header analysis, import/export parsing, section entropy, TLS callbacks, anti-analysis detection, overlay analysis.

YARA Scanning

Custom YARA rules for malware families, packers, ransomware, trojans, and scripts. Add your own rules easily.

ML Classification

Random Forest, XGBoost, and SVM models. Binary and multi-family classification with full metrics and anomaly detection.

Threat Intelligence

5 feeds: VirusTotal, MalwareBazaar, AlienVault OTX, AbuseIPDB, Shodan. Automatic IOC enrichment and correlation.

Capability Detection

Detects ransomware, keyloggers, C2 communication, credential stealing, evasion techniques, and persistence mechanisms.

IOC Graph & STIX

Interactive graph visualization of IOCs, relationships, and attack patterns. Export as STIX 2.1 bundles for sharing.

Batch Ingestion

Automated sample collection from MalwareBazaar feeds. Continuous ingestion with configurable delays and limits.

Family Detection

Multi-signal family detection combining YARA, import hashing, behavioral patterns, and ML similarity scores.

REST API & SDKs

Full REST API with JWT auth, API keys, rate limiting, OpenAPI docs, and webhook notifications for automation.
